FrameNet API 0.4 command line access tool

SYNOPSIS

       java -jar FrameNet.jar [Options] [Frames] [Frame Elements]

DESCRIPTION

  Displays information about the given frames or frame elements. The
  kind of information to show is controlled with options. The frames 
  to display are listed after the options. 
  If an asterisk (*) is given as an argument, all frames defined in 
  the frame database are given.

OPTIONS

$OPTIONS
